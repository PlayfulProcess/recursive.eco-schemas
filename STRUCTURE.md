# Repository Structure

This repository organizes grammars for symbolic systems — tarot, I Ching, astrology, literature, mythology, and more — in a way that nurtures the recursive public and the commons.

## Directory Organization

### Category Folders

Each category folder contains grammar subfolders (with `grammar.json` files) and optionally a `schemas/` subfolder with reference data and schema definitions.

- **`tarot/`** — Tarot decks and card oracle systems
  - Grammars: Rider-Waite, Alice's Tarot, Emerald Death Card visuals, Path of the Ontoject
  - `schemas/` — Reference data: Marseille, Etteilla, Sola Busca, Lenormand, anti-oppressive project, and more

- **`iching/`** — I Ching hexagram systems
  - Grammars: Chinese original with translations, HD Meta Categories (two versions)
  - `schemas/` — Reference data: Chinese summaries, Human Design mappings, AI summaries

- **`astrology/`** — Astrological interpretation systems
  - `schemas/` — L1 basic, Alan Leo, Jyotish Vedic, Ptolemy Tetrabiblos, Proctor, Bulfinch mythology

- **`classics/`** — Sacred, philosophical, and classical texts
  - Grammars: Confucian Analects (749 items), Dhammapada (431 items), Shakespeare's Ten Greatest (247 items)
  - `schemas/` — Dhammapada chapter schemas

- **`literature/`** — Fiction and literary works
  - Grammars: Alice in Wonderland, Winnie-the-Pooh

- **`mythology/`** — Myths, legends, and death traditions
  - Grammars: Greek Mythology, Bulfinch's Classical Mythology, Emerald Death Card (Egyptian/Tibetan/Gilgamesh)

- **`kids/`** — Children's content: stories, workouts, educational videos
  - Grammars: Shiryo Cat Stories, Art with Mati and Dada, Folk Tales, Greek Mythology for Kids, PBS Kids, Patchwork Pawls, GVO Kids Workouts, story sequences

- **`music/`** — Music and performing arts
  - Grammars: Luiza Lian (Brazilian music/poetry), Royal Ballet and Opera

- **`practice/`** — Social and therapeutic practice
  - Grammars: Social Working (decolonial deck), Family Therapy Models

- **`sequences/`** — Ordered video/content collections
  - Grammars: National Geographic Tucci in Italy

### Support Folders

- **`sources/`** — Public domain source texts used to build grammars (Shakespeare, Confucius, Dhammapada, Alice, Pooh, Zohar, astrology sources)
- **`future_plan/`** — Build logs, pipeline planning, grammar ideas, and principles

## Philosophy

This structure is designed to:

1. **Be Accessible**: Category-based organization makes it easy to find and contribute grammars
2. **Nurture Growth**: Separate spaces for different traditions allow each to flourish
3. **Support Commons**: Sources and schemas are shared resources for the community
4. **Enable Recursion**: The structure can grow and adapt as the community evolves
