# Data Directory

This directory contains data files that circulate throughout the recursive.eco ecosystem. These files can be exported and imported through the offer tab or flow app, enabling sharing and reuse across the ecosystem.

## Purpose

- Store example readings and interpretations that circulate in the ecosystem
- Share community-contributed data across offer and flow apps
- Provide training data for pattern recognition
- Document real-world usage of schemas
- Enable export/import functionality between ecosystem apps

## Guidelines

- Anonymize all personal information
- Include context files explaining the data format
- Respect privacy and obtain consent for shared readings
- Use descriptive filenames indicating content and date

## File Organization

Consider organizing by:
- Date/timestamp
- Reading type
- Divination system used
- Community event or project

Example: `2024-01-tarot-community-reading.json`
