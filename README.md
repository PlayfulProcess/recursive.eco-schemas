# recursive.eco-schemas

Where community's schemas of interpretation for tarot, I Ching, Astrology and Others live and nurture the recursive public.

## Purpose

This repository serves as a commons for interpretive schemas used in divination and self-reflection practices. It brings together structured data, templates, and community wisdom to support the recursive.eco ecosystem and the broader community.

## What's Inside

- **`/schemas/`** - Structured interpretive schemas for Tarot, I Ching, Astrology, and other systems
- **`/sources/`** - Public domain source materials for schema creation

## Quick Start

1. Browse the `/schemas/` directory to find interpretive frameworks
2. Check `/sources/` for public domain materials to reference
3. Read [CONTRIBUTING.md](CONTRIBUTING.md) to add your own schemas
