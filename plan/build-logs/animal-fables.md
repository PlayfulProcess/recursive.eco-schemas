# Build Log: Animal Fables — The Stories Nobody Finished

**Date:** 2026-03-12
**Grammar:** `grammars/animal-fables/grammar.json`
**Type:** tarot (oracle deck)
**Source:** Written from memory — structural bones from world epics, original retellings

## Concept

Anti-fables. Aesop uses animals to make morals simple. This grammar uses animals to make morals impossible. Each story: one animal, one minute, one situation with no clean answer. No moral at the end. Ever.

Oracle deck format — 25 cards, pull one at breakfast, talk about it or don't.

## Structure

- **25 L1 items** — individual fables, one animal each
- **5 L2 themes** — "Was It Worth It?", "Who Was Right?", "What Would You Have Done?", "Does It Ever End?", "What Did It Become?" (5 fables per theme)
- **1 L3 meta** — "The Stories Nobody Finished" (all themes)
- **31 total items**

## Design Decisions

- **Grammar type: tarot** — oracle deck format, pull-one-card interface
- **Sections: "Story" + "Sit With"** — minimal. No "Moral", no "Lesson", no "Shadow/Light". The "Sit With" is always a question, never an answer.
- **metadata.bone_from** — names the epic character/story each fable is structurally lifted from. A kid never needs to know this. An adult might enjoy it.
- **metadata.read_time: "1 minute"** — every fable is designed for sixty-second oral reading
- **No animal repeats** — 25 unique animals, one per fable
- **L2 themes are questions** — not categories. "Was It Worth It?" not "Sacrifice Stories"
- **L3 "How to Use" section** — explicit oracle deck instructions (pull at breakfast, etc.)

## Learnings

- Writing from memory (no source text parsing) is faster but requires disciplined consistency in tone and length
- The anti-moral format is harder than it looks — the temptation to sneak in a lesson is strong. Every "Sit With" was rewritten to end with a question, not an observation.
- 25 is the right number for a breakfast oracle. More feels like homework. Fewer feels thin.
- The Mahabharata provided 9 of 25 fables — it's the richest source for morally impossible situations
