# Family Therapy Grammar — Plan

**Grammar**: `grammars/family-therapy-grammar/grammar.json`
**Type**: from-memory (with links to public domain source texts)
**Grammar Type**: `custom`
**Target**: ~120–160 items across 3 levels
**Shelves**: `mirror`, `wisdom`, `resilience`
**Roots**: `emotion-love`, `eastern-wisdom`, `self-knowledge`, `process-philosophy`
**Lineages**: `Gottman`, `Johnson`, `Linehan`, `Andreotti`
**Worldview**: `dialectical`

---

## Vision

A grammar that maps the **living landscape of family therapy** — from the couple dyad to the child's inner world to the classroom to the body's trauma memory. Not a textbook. A relational map. Every item is a concept you can *draw* like a card: "What does Gottman's Four Horsemen say about what's happening in this room right now?"

This grammar is unique because it **bridges four domains that therapists usually silo**:

1. **Couples therapy** — what happens between two adults
2. **Child/play therapy** — what happens inside a child
3. **Pedagogy** — how we raise and teach children across cultures
4. **Trauma** — what the body remembers and how systems heal

The connecting thread is **attachment** — Bowlby's insight that we are wired for connection from birth to death, and that rupture and repair is the fundamental rhythm of all human relating.

## Section Structure

Each L1 item uses this section pattern:

```json
{
  "sections": {
    "Concept": "What this is. Clear, jargon-free explanation. 2-3 paragraphs.",
    "In the Room": "What this looks like in practice — in a therapy session, a classroom, a family dinner. Concrete, sensory, alive.",
    "The Research": "Key studies, names, dates. Not exhaustive — just enough to anchor.",
    "Questions": "3-5 reflective questions. For therapists, parents, teachers, or anyone in relationship."
  }
}
```

For **pedagogy** items, replace "In the Room" with:

```json
{
  "In the Classroom": "What this looks like in a learning environment. The teacher, the child, the space."
}
```

For **trauma** items, add:

```json
{
  "The Body": "What this feels like somatically. Where it lives in the nervous system."
}
```

## What Makes This Grammar Special

1. **It's a Rosetta Stone.** Gottman's "flooding" is Porges' "dorsal vagal." Johnson's "protest polka" is Bowlby's "anxious attachment" is the child's tantrum in the playroom. This grammar names those translations.

2. **It bridges clinic and classroom.** Most resources silo therapy and education. But Montessori was a doctor. Axline was a teacher. The best classroom IS therapy. The best therapy IS education.

3. **It honors non-Western approaches.** Indigenous pedagogy, Ubuntu, Gurukul, Vedic — these aren't "alternatives." They're often the originals. Western attachment theory is one lens among many.

4. **It's for parents, not just professionals.** The "Questions" sections are written for anyone in relationship — not just credentialed therapists.

5. **The cross-links make it alive.** When you draw "Bowlby's Attachment Theory" and it links to Darwin's *Expression of Emotions*, you see the lineage. When "Play Therapy" links to *Alice in Wonderland*, you feel the connection. The grammar breathes.
