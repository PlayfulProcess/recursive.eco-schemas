## Adding Sources

When adding a new source to `/sources/`:

1. Verify the work is in the public domain
2. Add an entry to this attribution file
3. Include the source URL and license in the schema's `attribution` field
4. Credit the translator/editor if applicable

When using a source for enrichment without storing the full text:

1. Add the source to the "Currently Used Sources" table
2. Include direct quotes in the schema's `quotes` array
3. Reference the source in the schema's `attribution` field
