# Templates Directory

This directory contains reusable templates for creating offers, flows, and ritual structures.

## Purpose

Templates help the community:
- Create consistent reading experiences
- Share ritual structures
- Build new flows and offers
- Maintain quality standards

## Types of Templates

- **Reading Templates**: Structured formats for different types of readings
- **Offer Templates**: Formats for creating community offers
- **Flow Templates**: Patterns for guiding users through interpretive processes
- **Ritual Templates**: Structures for group or solo practices

## Using Templates

1. Copy the template file
2. Fill in the specific content
3. Customize for your context
4. Share back improvements to the template itself

## Contributing Templates

When contributing templates:
- Make them flexible and adaptable
- Include clear instructions and comments
- Document required vs. optional parameters
- Provide at least one example usage

See [CONTRIBUTING.md](../CONTRIBUTING.md) for more details.
