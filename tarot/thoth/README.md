# Thoth Tarot

This directory is reserved for the Thoth Tarot deck tradition.

## About Thoth Tarot

The Thoth Tarot is a tarot deck designed by Aleister Crowley and painted by Lady Frieda Harris. It was published posthumously in 1969. The deck is based on Crowley's "Book of Thoth" and incorporates elements from Thelema, astrology, Kabbalah, and other esoteric systems.
